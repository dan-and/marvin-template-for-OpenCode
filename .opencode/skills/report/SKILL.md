## What I Do
- Synthesizes work across all sessions from the past week
- Generates comprehensive weekly summary reports
- Tracks topics, projects, and productivity metrics

## When to Use
Run this skill at the end of your week to create a comprehensive report of your progress and work accomplished.

## Steps
1. Review all session logs from the past 7 days
2. Extract key projects and topics worked on
3. Calculate session metrics (count, duration)
4. Synthesize deliverables and accomplishments
5. Generate formatted weekly report
6. Save report to reports/ directory

## Output
A markdown file with:
- Session summary and metrics
- Project and topic breakdown
- Accomplishments and deliverables
- Next week priorities
- Velocity and trends analysis
